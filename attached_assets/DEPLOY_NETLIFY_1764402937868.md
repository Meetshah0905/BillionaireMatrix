## Method 1: Deploy via Netlify Website (Easiest)

### Step 1: Build Your App
1. Open terminal/command prompt in your project folder
2. Run:
   ```bash
   npm install
   npm run build
   ```
3. This creates a `dist/public` folder with all your files

### Step 2: Deploy to Netlify
1. Go to [https://app.netlify.com](https://app.netlify.com)
2. Sign up or log in (it's free!)
3. Click **"Add new site"** â†’ **"Deploy manually"**
4. Drag and drop the `dist/public` folder onto the deployment area
5. Wait for upload to complete
6. Your site will be live at `https://your-site-name.netlify.app`

- Open your mobile browser (Safari, Chrome, etc.)

## Method 2: Deploy via Git (Recommended for Updates)

### Step 1: Push to GitHub
1. Create a GitHub repository
2. Push your code:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin YOUR_GITHUB_REPO_URL
   git push -u origin main
   ```

### Step 2: Connect to Netlify
1. Go to [https://app.netlify.com](https://app.netlify.com)
2. Click **"Add new site"** â†’ **"Import an existing project"**
3. Choose **GitHub** and authorize Netlify
4. Select your repository
5. Configure build settings:
   - **Build command:** `npm run build`
   - **Publish directory:** `dist/public`
6. Click **"Deploy site"**

### Step 3: Automatic Updates
- Every time you push to GitHub, Netlify automatically rebuilds and deploys

## Troubleshooting

### Build Fails?
- Make sure you ran `npm install` first
- Check that `package.json` has all dependencies

### Site Not Loading?
- Check the publish directory is `dist/public`
- Verify build completed successfully

### Mobile Not Working?
- Clear browser cache
- Make sure you're using the Netlify URL (not localhost)
- Try a different mobile browser
