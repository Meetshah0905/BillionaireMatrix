# Billionaire Matrix (Minimalist)

A minimalist, table-first 2x2 matrix for task classification based on Energy vs. Money.

## Features

- **Minimalist UI**: Clean table layout, no distractions.
- **Smart Suggestions**: Offline heuristic engine suggests quadrant placement based on keywords.
- **Learning System**: Remembers your overrides to improve future suggestions.
- **Local Storage**: Data stays on your device.
- **Export/Import**: Backup your data to JSON.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Run tests:
   ```bash
   npm test
   ```

## Architecture

- **Frontend**: React + Vite
- **State**: Zustand (persisted to localStorage)
- **Styling**: Tailwind CSS
- **Logic**: `client/src/lib/suggestionEngine.ts` (Pure function classifier)

## Matrix Logic

- **PROTECT**: Gives Energy + Takes Money
- **PRIORITIZE**: Gives Energy + Makes Money
- **DELETE**: Takes Energy + Takes Money
- **DELEGATE**: Takes Energy + Makes Money
